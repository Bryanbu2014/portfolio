Jetson Self Driving Robot Car

Autonomous driving and obstacle avoidance using Computer Vision.

View Source View Result

Introduction

This is the project in module "Computer Vision" in 4th semester of my course, Artificial Intelligence. The goal of this project is to program an autonomous driving robot with computer vision techniques using Jetson Nano Developent Kit from Waveshare. The desired functions of this autonomous driving robot are road following and obstacle avoidance, which means the robot can drive by itself in a designed track and stop before it hits an obstacle.

Motivation: Autonomous Driving

  • Many groundbreaking and innovative inventions in the field of Computer Vision (CV)
  • Interested to see how the theories we learned are translated into code
  • How different each AI models process and predicts images

Features

  • Stage 1 (road following) → To drive along the designated track
  • Stage 2 (collision avoidance) → To avoid obstacles as the robot drives along the designed track i.e. The robot stops if there is an obstacle ahead

Methodology

Workflow is concluded generally as follow:

  • Data collection
  • Train a model with the data collected
  • Loading the model on the Jetbot

Result / Demonstration

In this video, the Jetson robot car successfully drives itself around the designated track using its camera to follow the lines. It also detects obstacles in its path and stops automatically to avoid a crash.

Lesson Learned – In a Nutshell

  • Hardware: Understanding how to work with limited resources (Robot CPU is the biggest hardware limitation, Motor limitations).
  • Software: Understand how to read software parameters through documentations (Libraries break apart easily, Third party resources, Libraries are constantly updating).
  • Environment: Understand how the contrast in environment plays a role (High contrast environment is recommended, Too much light can interfere with decision making).
