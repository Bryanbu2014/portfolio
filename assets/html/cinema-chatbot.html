Cinema Chatbot Github

This is the project that was done during my course of study. In this project, a friend and I were asked to develop a simple chatbot using the Rasa framework.

There were initially a few ideas that came to our mind, such as "Munich Travel Partner" and "School Canteen Assistant." But in the end, we decided to go with the "Cinema Chatbot Assistant," which could be a website or an application.

Cinema Chatbot

Yeap, from the name itself, I believe you can already tell; it's a chatbot that assists customers in every process at the cinema. Users are expected to ask the chatbot anything related to the cinema, such as:

  • Descriptions of movies
  • Which movie is available on a particular day
  • The seating plan
  • The show schedule
  • Ticket prices

Of course, we didn't actually have access to a real database, so we designed and used our own "mini-database," which consists of only very little movies and their information.

For more technical details, please refer to my Github repository.
